# Palo Alto PCNSA

**Certificação:** Palo Alto Networks Certified Network Security Administrator  
**Custeamento:** Kyndryl (Costed Learning)

## Descrição
Certificação de segurança de rede enterprise com firewalls Palo Alto Networks. Complemento ao NSE 4 da Fase 3 — cobrindo um segundo vendor de referência no mercado enterprise, utilizado internamente na Kyndryl.

## Formação
- Palo Alto Networks EDU-110 — Firewall Essentials: Configure and Manage (oficial)
- Palo Alto Networks Learning Center

## Conteúdo abordado
- PAN-OS — arquitectura e administração
- Security policies e App-ID — identificação de aplicações
- User-ID e content-ID — controlo baseado em utilizador e conteúdo
- VPN — GlobalProtect e site-to-site IPsec
- Threat prevention — IPS, anti-malware, DNS Security
- Logging, monitoring e Panorama

## Estrutura
- `notas/` — conceitos, políticas e configurações Palo Alto
- `labs/` — exercícios práticos documentados
