# Linux Security — EX415

**Certificação:** Red Hat Certified Specialist in Security: Linux  
**Custeamento:** Kyndryl (Costed Learning)

## Descrição
Hardening avançado de sistemas Red Hat Enterprise Linux — SELinux, AIDE, LUKS, OpenSCAP e PAM. Um dos exames mais relevantes para um perfil DevSecOps com foco em segurança de infraestrutura.

## Formação
- RH415 — Red Hat Security: Linux in Physical, Virtual, and Cloud (formação oficial)
- Documentação oficial Red Hat — Security Guide for RHEL

## Conteúdo abordado
- SELinux — políticas, contextos, troubleshooting avançado
- OpenSCAP — auditorias de conformidade e remediação automática
- AIDE — detecção de intrusão baseada em ficheiros
- LUKS — encriptação de discos e volumes
- PAM — autenticação pluggable, políticas de password
- Firewalld avançado — zonas, serviços, rich rules
- Auditd — auditoria de eventos de sistema

## Estrutura
- `notas/` — conceitos, comandos, políticas de segurança
- `labs/` — exercícios práticos documentados
