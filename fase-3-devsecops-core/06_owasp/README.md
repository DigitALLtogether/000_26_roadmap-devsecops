# OWASP — Segurança Aplicacional

**Objectivo:** Integração de segurança no ciclo de desenvolvimento (DevSecOps)  
**Custeamento:** Gratuito (OWASP, TryHackMe parcial)

## Conteúdo abordado
- OWASP Top 10 (aplicações web)
- OWASP API Security Top 10
- Threat Modeling — metodologia STRIDE + OWASP Threat Dragon
- SAST: SonarQube
- DAST: OWASP ZAP
- SCA: Trivy, Snyk
- TryHackMe SOC Level 1 (~€14/mês)

## Artefacto de portfólio
- Pipeline completo SAST+DAST+SCA em GitHub Actions
- Threat model com diagramas STRIDE
- Análise OWASP API Security numa API real

## Estrutura
- `notas/` — conceitos de segurança aplicacional
- `labs/` — exercícios práticos e análises documentadas
