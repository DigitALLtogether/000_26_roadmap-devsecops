# GitHub Advanced Security

**Certificação:** GitHub Certified — Advanced Security  
**Custeamento:** Kyndryl (Costed Learning)

## Formação
- GitHub Learning Lab — GitHub Advanced Security
- GitHub Docs — Code scanning, secret scanning, dependency review

## Conteúdo abordado
- Code scanning com CodeQL — detecção de vulnerabilidades no código
- Secret scanning — detecção automática de credenciais expostas em repositórios
- Dependency review — análise de dependências em pull requests
- Security advisories — gestão de vulnerabilidades descobertas
- SARIF — formato de relatórios de análise estática
- Integração com GitHub Actions para segurança contínua

## Estrutura
- `notas/` — conceitos, configuração e análise de resultados
- `labs/` — configuração prática e análise de resultados documentada
