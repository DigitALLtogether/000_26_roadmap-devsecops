# Segurança — Fortinet NSE 4

**Certificação:** NSE 4 — FortiGate Security  
**Custeamento:** Kyndryl (parceiro Fortinet)

## Descrição
Primeiro NSE com exame formal — complementa os NSE 1+2+3 da Fase 1 com competências técnicas práticas. Configuração e gestão de FortiGate: firewall, NAT, VPN, IPS e web filtering. Relevante para o perfil DevSecOps com foco em segurança de rede enterprise.

## Formação
- Fortinet NSE Training Institute — NSE 4 FortiGate Security (oficial, online)
- Formação interna Kyndryl (parceiro Fortinet)

## Conteúdo abordado
- Firewall policies — criação e gestão de regras
- NAT — source e destination NAT
- VPN — SSL-VPN e IPsec site-to-site
- IPS — Intrusion Prevention System
- Web filtering e application control
- Logging e reporting FortiGate

## Estrutura
- `notas/` — conceitos, políticas e configurações FortiGate
- `labs/` — exercícios práticos documentados
