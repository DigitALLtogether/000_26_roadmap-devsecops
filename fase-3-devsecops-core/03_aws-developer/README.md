- Lambda — serverless compute, event-driven architecture
- API Gateway — REST e HTTP APIs
- DynamoDB — NoSQL, operações e optimização
- SQS, SNS, EventBridge — mensageria e eventos
- CodePipeline, CodeBuild, CodeDeploy — CI/CD nativo AWS
- X-Ray — tracing e observabilidade de aplicações
- IAM — credenciais e autenticação para developers
