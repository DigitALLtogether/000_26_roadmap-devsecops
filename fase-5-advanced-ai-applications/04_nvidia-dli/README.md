# Nvidia DLI Courses

**Tipo:** Formação técnica certificada por curso  
**Custeamento:** Kyndryl (parceiro Nvidia)

## Formação
- Nvidia Deep Learning Institute (DLI) — cursos seleccionados via Kyndryl
- Foco: GPU computing, inferência e IA aplicada a operações de segurança

## Cursos relevantes (seleccionados)
- Fundamentals of Deep Learning
- Building Transformer-Based NLP Applications
- Deploying a Model for Inference at Production Scale (Triton)
- Building AI Applications with Vector Databases

## Aplicação ao percurso DevSecOps
Fundamentos técnicos para os projectos de portfólio da Fase 5 — AI Security Copilot e Anomaly Detection Pipeline. Nvidia DLI fornece o substrato de GPU computing e inferência em produção que complementa a formação AWS/GCP.
