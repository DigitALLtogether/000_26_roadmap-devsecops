# Projectos — IA Aplicada

Projectos práticos que combinam IA com DevSecOps.

## Projectos planeados

### AI Security Copilot
Sistema baseado em LLM para triagem automática e sumarização de alertas de segurança.

### Anomaly Detection Pipeline
Pipeline de machine learning para detecção de anomalias em logs de segurança.

### AI Code Review in CI/CD
Integração de LLM em pipeline CI/CD para análise automática de Pull Requests.

## Estrutura
- `notas/` — investigação, arquitectura e decisões de design
- `labs/` — experimentação e protótipos
- `scripts/` — código Python dos projectos
