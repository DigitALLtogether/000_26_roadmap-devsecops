# Incident Response

**Objectivo:** Competências de resposta a incidentes e análise forense  
**Custeamento:** Gratuito (BlueTeamLabs, SANS Cyber Aces)

## Formação
- BlueTeamLabs — exercícios práticos de IR e forense
- SANS Cyber Aces (gratuito)

## Artefacto de portfólio
- IR playbook com simulação completa de incidente

## Estrutura
- `notas/` — fases IR: Preparação, Detecção, Contenção, Erradicação, Recuperação
- `labs/` — simulações de incidentes documentadas
- `playbooks/` — runbooks de resposta a incidentes
