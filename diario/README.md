# Diário de Estudo

Registo semanal do percurso — o que estudei, o que pratiquei, o que ficou por fazer.

## Formato dos ficheiros
`AAAA-WNN-tema.md`  
Exemplo: `2026-W23-inicio-percurso.md`

## Para que serve
- Manter consciência do progresso real
- Identificar padrões (semanas boas vs. fracas)
- Servir de contexto para sessões de estudo seguintes
