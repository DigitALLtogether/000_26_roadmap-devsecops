# Red Hat Containers — EX188

**Certificação:** Red Hat Certified Specialist in Containers  
**Custeamento:** Kyndryl (Costed Learning)

## Descrição
Gestão de containers OCI sem Docker daemon — Podman, Buildah e Skopeo. Complementa o módulo Docker com a abordagem Red Hat enterprise (rootless, daemonless).

## Formação
- RH134 — Red Hat System Administration II (cobre Podman base, já incluído no RHCSA)
- Documentação oficial Red Hat — Container Tools Guide

## Conteúdo abordado
- Podman — gestão de containers rootless e rootful
- Buildah — construção de imagens OCI sem Dockerfile
- Skopeo — inspecção e cópia de imagens entre registos
- Podman Compose — alternativa ao Docker Compose
- Integração com OpenShift (base para EX280)

## Estrutura
- `notas/` — conceitos, comandos essenciais, diferenças Podman vs Docker
- `labs/` — exercícios práticos documentados
